\section{Quickstart}\label{quickstart}

The fastest way to get beats out of your new box:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Download the factory SD card image from:
  \url{http://sonic-potions.com/public/SdCardImage.zip}
\item
  Unzip all files to the root folder of your (FAT32!) SD card.
\item
  Insert the card and power up the synth.
\item
  Press the Play button. The chaselight starts running.
\item
  Use the 16 step buttons to activate some steps. You should now hear
  the first voice playing.
\item
  Use the 7 voice select buttons to change the active track/voice.
\item
  Again, use the 16 step buttons to activate some steps. The next voice
  should now play as well.
\item
  Press the Load/Save mode button once. You are now in preset load mode.
\item
  Use the encoder to select another preset. The sound changes as soon as
  you move the encoder.
\end{enumerate}
